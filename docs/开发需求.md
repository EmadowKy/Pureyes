# Pureyes 项目开发需求文档

## 📋 项目概述

**Pureyes** 是一个视频 AI 问答系统，支持上传视频后对视频内容进行智能问答。

### 技术栈
- **后端**: Python + Flask (端口 8000)
- **前端**: Vue3 + Vite (端口 5173)
- **AI 模型**: PyTorch + Transformers (多模态视频分析)
- **数据库**: SQLite

### 项目结构
```
Pureyes/
├── backend/
│   ├── app/
│   │   ├── core/       # 核心配置、数据库、响应工具
│   │   ├── mva/        # 多模态视频分析 (agent.py, agent_runner.py, utils.py)
│   │   ├── qa/         # 问答模块 (routes.py, qa_manager.py, run_model.py)
│   │   ├── video/      # 视频模块 (routes.py, views.py)
│   │   └── video_routes.py  # 视频播放路由
│   ├── configs/        # 配置文件
│   ├── output/         # 输出目录
│   └── run.py          # 启动入口
├── frontend/
│   └── src/
│       ├── api/        # API 调用
│       ├── components/ # 组件 (qa/)
│       ├── views/      # 页面视图 (IntegratedQA.vue, qa/, video/)
│       ├── router/     # 路由配置
│       ├── App.vue
│       └── main.js
└── docs/
```

## 🎯 当前开发任务

### 任务 1: 完善前端问答界面

**文件**: `frontend/src/views/IntegratedQA.vue`

**需求**:
1. 检查当前界面功能完整性
2. 优化用户交互体验
3. 确保与后端 API 正确对接
4. 添加加载状态和错误处理

**API 端点**:
- `POST /api/qa/ask` - 提交问题
- `GET /api/qa/history` - 获取历史记录
- `GET /api/qa/task/<task_id>/status` - 查询任务状态

## ❓ 常见问题

如果遇到依赖问题：
```bash
source ~/bin/activate
pip install -r requirements.txt --force-reinstall
```

如果前端依赖问题：
```bash
cd frontend
rm -rf node_modules package-lock.json
npm install
```
